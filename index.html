Programming Language Research with Artificial Intelligence

This page presents three complementary projects where I explored Programming Language research with AI support: from language design and tooling ideas to practical implementations and educational resources.

AspectScript

A research-oriented JavaScript extension for aspect-oriented programming, focused on expressive pointcuts, advice composition, and advanced control over cross-cutting concerns.

Visit project GitHub repository

ESAJS

A practical evolution of the AspectScript direction, emphasizing a cleaner and usable implementation path for experimenting with aspect-based techniques in modern JavaScript environments.

Visit project GitHub repository

Continuations in Go

A continuation-focused exploration in Go that investigates control flow abstractions and runtime behavior, helping bridge theory and engineering practice in PL concepts. DeloreanGo extends this line of work by bringing the previous DeloreanJs ideas from JavaScript into a new implementation for Go.

Visit project GitHub repository
